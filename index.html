Projects I've Created

Flashcards For To Learn Stuff

This Flashcards project utilizes, HTML, CSS, ES6/JavaScript and React to create an application capable of creating, displaying, using, and quizzing on flashcard sets. It comes pre-loaded with a few decks for demo purposes but feel free to create more decks! Easily review card sets and try one of the three different quiz varieties: multiple choice answer, write-in answer, and self-survey!

In-Browser Drum Kit
